<h2 align="center">获取新闻热榜展示</h2>

- 需要在linux(代码编辑器)等终端中运行体验最佳，已支持接口方式调用及查看

- 安装依赖 ` npm i`

- 运行 `node app`

输入 http://localhost:8090 进入首页 即可查看(摸鱼) 最新的热榜消息
  
- 目前对接了：
- `百度热榜`
- `微博热榜信息`
- `0818团线报`
- `yaohuo(需自行配置cookie)`
- `线报屋-豆瓣`
