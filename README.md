<h2 align="center">获取新闻热榜展示</h2>

- 需要在linux(代码编辑器)等终端中运行体验最佳

- 安装依赖 ` npm i`

- 运行 `node app`

- 目前对接了百度热榜、微博热榜信息、0818团线报、yaohuo列表。

- 即可查看(摸鱼) 最新的热榜消息
